# webpack5-vue3

> webpack5-vue3

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for local with minification
npm run build:local

# build for ldev with minification
npm run build:ldev

# build for test with minification
npm run build:test

# build for prod with minification
npm run build:prod

# build for production and view the bundle analyzer report
npm run build:local --report
npm run build:ldev --report
npm run build:test --report
npm run build:prod --report
```
